წასაკითხი ვებ აპლიკაცია - თავი 19

Древоხე зимоюზამთარში пред окнамиფანჯრებწინ келлииკელიისა

Зимуზამთარი 1828-го1828 годаწლისა яმე провелგავატარე в монастыреმონასტერში преподобногоღირსი Александраალექსანდრე Свирскогоსვირელისა . Предწინაშე окнамиფანჯრებისა моейჩემი келлииკელიისა стоялоიდგა древо,ხე, разоблаченноеგაშიშვლებული морозами,ყინვებისაგან, какროგორც скелет,ჩონჩხი, разоблаченныйგაშიშვლებული смертью.სიკვდილის მიერ. Уединениеგანმარტოვება изощряетამახვილებს чувства,შეგრძნებებს, изощряетამახვილებს мысль;ფიქრს; кругწრე действияქმედებებისა ихმათი расширяется.ფართოვდება. Между темმათ შორის море,ზღვა, о которомრომელზეც святойწმინდა Иоаннიოანე Лествичникკიბისაღმწერელი говорит,ამბობს, чтоრომ емуიგი непременноაუცილებლად должноუნდა взволноваться,აღელვებულიყო, волновалосьღელავდა 101 . Обнаженноеგაშიშვლებული древоხე служилоემსახურებოდა для меняჩემთვის утешением:ნუგეშს: оноიგი утешалоმანუგეშებდა меняმე надеждоюიმედით обновленияგანახლებისა душиსულისა моей.ჩემისა «Гласомჴმითა моим»,ჩემითა,“ гласомხმით умаგონებისა моего,ჩემისა. гласомხმით сердцаგულისა моего,ჩემისა. гласомხმით телаსხეულისა моегоჩემისა болезнующего,ავადმყოფისა, гласомხმით немощейუძლურებათა моих,ჩემთა, гласомხმით паденийდაცემებისა моихჩემთა "воззвах"ღაღად-ვყავ“ ( Пс.141:2ფს. 141:2 ). «Господиუფალო, услышиშეისმინე молитвуლოცვისა мою»ჩემისა“ ( Пс.142:1ფს. 142:1 ), «вонмиმოხედენ молениюვედრებასა моему»ჩემსა“ ( Пс.141:7ფს. 141:7 ), котороеრომელსაც воссылаюაღმოგიგზავნი Тебеშენ из средыგარემოდან браней,ბრძოლებისა, потрясающихშემარყევლებისა умგონებისა мойჩემისა иდა сердце,გულისა, из средыგარემოდან болезней,სნეულებებისა томящихგამტანჯველისა иდა расслабляющихდამაუძლურებლისა телоსხეულისა мое,ჩემისა из средыგარემოდან множестваმრავალი немощей,უძლურებათა объемлющихმომცველთა всеმთელი существованиеარსებისა мое,ჩემისა, из средыგარემოდან бесчисленныхაურაცხელი падений,დაცემებისა, которымиრომლებითაც преисполненаაღსავსეა жизньცხოვრება моя.ჩემი. Услышавшийგამგონებელო Иону,იონისაო, молившегосяმლოცველისა во чревеმუცელში кита,ვეშაპისა, услышьგამიგონე меня,მეც, вопиющегоმგოდებელისა из чреваმუცლიდან беззаконийუშჯულოებათა моих,ჩემთა, из чреваმუცლიდან адова.ჯოჯოხეთისა. Из глубины,სიღრმიდან, из бездныუფსკრულიდან грехов,ცოდვებისა из бездныუფსკრულიდან поползновенийდაცემებისა иდა искушенийგანსაცდელებისა моихჩემთა «воззвахღაღად-ვყავ Тебе,შენდამი, Господи!»უფალო!“ Господи,უფალო услышьისმინე гласხმისა мой!ჩემისა! «Изведиგამოიყვანე из темницыსაპყრობილით страстейვნებებისა душуსული мою",ჩემი“ пролейჩაღვარე в нееმასში светნათელი благодатный!მადლისმიერი! ( Пс.141:8ფს.141:8 ). Когдаროდესაც прольешьჩაღვრი в нееმასში этотამ свет,ნათელს, светნათელს и светлый,ნათელსა, иდა радостный,მხიარულს, иდა животворный,მაცოცხლებელს, тогдаმაშინ будетდაიწყებს онаიგი «исповедатисяაუარო имениსახელსა Твоему»შენსა“ ( Пс.141:8ფს. 141:8 ). Действуетმოქმედებს в душеსულში исповедание,აღიარება, возбужденноеაღძრული благодатью,მადლით, превышающееაღმატებული умგონებასა иდა потопляющееჩამძირავი егоმისი в неизреченнойგამოუთქმელ сладостиსიტკბოში своей;თავისაში; он,იგი, сошедшиჩამოსული в сердечнуюგულის клеть,საკუჭნაოში, затворившисьგამოკეტილი в нейმასში невниманиемუყურადღებობით ко всемуყველაფერი чувственному,გრძნობადისადმი произноситწარმოთქვამს имяსახელს Твое,შენსას, поклоняетсяთაყვანს სცემს имениსახელს Твоему,შენსას питаетсяიკვებება именемსახელით Твоим,შენით, объемлетმოიცავს имяსახელს Твоеშენსას иდა объемлетсяმოცულ იქნება им.მისით. Имяსახელი Твое,შენი Словеსიტყვაო Божийღვთისაო иდა Боже,ღმერთო, соделываетხდის для негоმისთვის излишнимиზედმეტს всеყველა прочиеსხვა слова!სიტყვას! «Избавиმიჴსენ мяმე от гонящихმდევართა мя»ჩემთაგან“ бесовскихეშმაკეული помысловგულისთქმებისგან иდა начинаний,წამოწყებებისაგან, «якоრამეთუ укрепишесяგანძლიერდეს паче мене»ჩემსა უფროს“ ( Пс.141:7ფს. 141:7 ), пачеუფროს произволенияნება-მყოფლობისა душиსულისა моей,ჩემისა, пачеუფროს постиженияშეგნებისა умаგონებისა моего!ჩემისა! «Уныმოეწყინა во мнеჩემ თანა духსულსა мой,ჩემსა, во мнеდა ჩემ შორის смятесяშემიძრწუნდა сердцеგული мое»ჩემი“ ( Пс.142:4ფს. 142:4 ). «Объяшаგარემომადგეს мяმე болезниსალმობანი смертные,სიკუდილისანი, бедыდა ჭირთა адовыჯოჯოხეთისათა обретошаმპოვეს мя»მე“ ( Пс.114:3ფს. 114:3 ).

Господи!უფალო! неარა на моюჩემს силуძალებზე уповаю:ვიმედოვნებ: паденияდაცემანი моиჩემნი научаютმასწავლიან меняმე познаватьშეცნობას немощьუძლურებისას мою.ჩემისას. Ты,შენ ხარ, Господи,უფალო, упованиеსასოება мое!ჩემი Тогдаმაშინ толькоმხოლოდ могуშემიძლია бытьვიყო «в стране,სოფელსა შინა живых»ცხოველთასა.“ ( Пс.114:9ფს. 114:8 ), в странеსოფელში святыяწმინდა правдыსიმართლისა Твоея,შენისა, когдаროდესაც Ты,შენ Господи,უფალო ниспошлешьგარდამოავლენ в сердцеგულში моеჩემსაში благодатьმადლს Твою,შენსას когда,როდესაც вселившисьდამკვიდრებული в сердцеგულში мое,ჩემსაში будешьიქნები «частьნაწილ... достоянияმკჳდრობისა моего»ჩემისა“ ( Пс.15:5ფს. 15:5 ), моимჩემი единственнымერთადერთი имуществомქონება иდა сокровищем!საგანძური! Возрадуютсяგაიხარებენ святыеწმინდა Ангелыანგელოზები Твои,შენნი, возрадуютсяგაიხარებენ ликиსახენი благоугодившиеსათნომყოფელთა Тебеშენდამი человеков,ადამიანებისა увидяდაინახავენ რა спасениеხსნას мое.ჩემსას «Менеმე ждутმელიან праведницы,მართალნი, Дóндежеვიდრემდის воздасиმომაგო мне»მე“ ( Пс.141:8ფს. 141:8 ) милостьწყალობა Твоюშენი неარა по множествуსიმრავლის მიხედვითაა греховცოდვებისა моих,ჩემთა, ноარამედ по множествуსიმრავლის მიხედვით Твоегоშენი человеколюбия.კაცთმოყვარეობისა. Аминь.ამინ.
